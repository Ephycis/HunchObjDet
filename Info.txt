This website currently just adds pictures that it can recognize later

Accessing webcam can only be done with https

Making another website:
$ heroku login
$ cd (my-project)/
$ git init
$ heroku git:remote -a (my-project)

To connect it to a github repository
$ git remote add origin https://github.com/(link it gives me)

The one it gave me:
$ git remote add origin https://github.com/Platt-Hunch/plattobjdetwebpage.git

The bare minimun to copy at make other websites:
index.js
package.json
Procfile
README.md

While in (my-project), to upload the changes:
$ git add .
$ git commit -m "(describe the changes)"
$ git push heroku master

Push to Github:
$ git add .
$ git commit -m "(describe the changes)"
$ git push -u origin master //For first time
$ git push //Every other push
